基本类型 #

Int

Double
Swift能够安全的存储15～16位数的Double类型，例如：

// _ 数字中 _ 会被自动忽略
let a: Double = 123_456_789_012_345

Decimal
精度要求较高时使用，比如涉及金钱。
Swift能够安全的存储38～39位数的Decimal类型，初始化时注意，通过Int\Double转换过来可能会有误差，通过String则更为精准。

String

Bool

Tuple

let allen = (name: "allen", height: 170)
let (_, height) = allen
print("allen height", height)
